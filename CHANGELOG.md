# 更新日志

本项目的每个公开版本都会在这里记录主要调整，并在 GitHub Releases 中附带对应 Android 安装包。

## [0.5.3] - 2026-07-19

- 新增“成绩”页：按学期查看课程总评、学分、绩点与成绩性质。
- 同一次教务登录会话会同步课表与学生个人成绩；成绩数据及同步快照只保存在当前设备。
- 单科成绩详情按需请求，支持展示教务系统提供的平时、期中、实验、期末、作业和占比等成绩组成。
- 成绩接口暂时不可用时，已同步的课表、成绩和历史快照都会保留。

## [0.5.2] - 2026-07-19

- 新增原生 iOS XCUITest 冒烟测试，自动检查首页、完整课表、课程添加、同步隐私说明和设置页面。
- iOS GitHub Actions 会自动选择并启动可用的 iPhone Simulator，在干净环境中执行界面交互测试。
- 每次测试都会上传页面截图、Xcode 测试结果包、构建输出和应用日志，失败时也能保留诊断材料。
- 版本号更新至 `0.5.2`，Android Release 继续由版本标签自动生成正式签名 APK。

## [0.5.1] - 2026-07-19

- Android 安装包改为正式 Release 构建，使用长期发布密钥签名，不再发布 Debug APK。
- GitHub Releases 自动构建流程接入加密签名密钥，版本标签会生成可直接安装的正式 APK。
- 教务同步增加首次隐私确认，明确应用为非官方学生工具，并说明密码、学号、课表和同步记录的数据处理方式。
- 设置页新增“隐私与非官方说明”入口，版本号同步更新至 `0.5.1`。

## [0.5.0] - 2026-07-19

- 新增完整的 Capacitor iOS 原生工程，与 Android 共用课程表、教务同步、本地存储、深色模式和课程时间设置。
- 配置 iOS 应用名称“课序”、应用标识 `com.campusflow.schedule`、统一图标与 `0.5.0` 版本信息。
- 增加 iOS 同步和 Xcode 构建命令，并完善 Pods、签名文件与构建产物的忽略规则。
- 将移动端工程统一命名为 `kexu-mobile`，版本页不再只显示 Android 平台。

## [0.4.1] - 2026-07-19

- 新增“每小节时长”设置，默认 45 分钟，可按 5 分钟在 30–60 分钟之间调整。
- 每两节大课内部保留 5 分钟小课间，上午第 2 节和下午第 6 节结束后保留 20 分钟长课间。
- 调整默认节次时间：1–2 节 08:15 开始，5–6 节 13:30 开始，9–10 节为 17:45–19:20，11–12 节为 19:30–21:05。
- 课程详情新增实际起止时间，首页上课状态判断与周课表时间轴会随课时长度同步更新。

## [0.4.0] - 2026-07-19

- 新增七天十二节的周课表网格，可直观看到课程时间与位置。
- 保留按星期分组的课程列表，支持通过右上角按钮切换两种课表布局。
- 自动记住用户选择的课表布局。
- 重绘教务同步页面的学期选择界面，改为适合手机操作的底部面板。
- 优化 360px 与 390px 手机屏幕适配，并继续突出显示已结课课程。
